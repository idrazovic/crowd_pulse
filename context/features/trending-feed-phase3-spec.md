# Feed UI Phase 1 Spec

## Overview

This is phase 3 of the trending feed UI layout. Use the screenshot referenced below for how it should look.

## Requirements for phase 3

- Remove search - icon, search bar,etc.
- Top bar needs to be present on every page - it is a part of layout that should always be present
- Vibe dropdown  needs to have the same look and feel as presented in @context/screenshots/trending-feed-vibe-dropdown.png - but leave the icons as they are in the code.
- Country dropdown needs to have the same look and feel as presented in  @context/screenshots/trending-feed-contry-dropdown.png
