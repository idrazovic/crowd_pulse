# CLAUDE.md

**CrowdPulse** is a community platform where users craft and vote on DJ lineups for real venues. Users pick a venue (via Google Places), a vibe tag, and an ordered DJ list (via Spotify). Lineups are public and ranked by upvotes.

## Commands

```bash
npm run dev        # start dev server (uses Turbopack by default)
npm run build      # production build
npm run start      # start production server
npm run lint       # run ESLint (build no longer runs the linter automatically)
npm run lint -- --fix  # auto-fix lint issues
```

There are no tests yet. Type-check with:
```bash
npx tsc --noEmit
```
